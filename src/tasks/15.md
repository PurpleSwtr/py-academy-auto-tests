  <h1 class="question-title">
   Нахождение Медианы
  </h1>
 </div>
 <div class="question-text">
  <p>
   В статистике медиана — это значение, которое разделяет верхнюю и нижнюю половины выборки данных. Для набора данных её можно рассматривать как "среднее" значение при сортировке данных.
  </p>
  <p>
   Ваша задача — реализовать функцию
   <span class="code code-line code-line--blue-color">
    find_median
   </span>
   , которая вычисляет медиану заданного набора чисел.
  </p>
  <h2>
   Формат ввода
  </h2>
  <p>
   Функция получает строку, содержащую целые числа, разделенные пробелами.
  </p>
  <h2>
   Формат вывода
  </h2>
  <p>
   Функция должна возвращать:
  </p>
  <ul>
   <li>
    Медианное значение чисел. Если результат является целым числом, верните целое число; в противном случае — число с плавающей точкой, округленное до 1 знака после запятой.
   </li>
  </ul>
  <p>
   Медиана вычисляется следующим образом:
  </p>
  <ul>
   <li>
    Если набор имеет нечетное количество элементов, медианой является средний элемент после сортировки.
   </li>
   <li>
    Если набор имеет четное количество элементов, медианой является среднее арифметическое двух средних элементов после сортировки.
   </li>
  </ul>
  <h2>
   Ограничения
  </h2>
  <ul>
   <li>
    Входная строка будет содержать от 1 до 1000 целых чисел.
   </li>
   <li>
    Каждое целое число будет в диапазоне от [-10^6, 10^6].
   </li>
  </ul>
  <h2>
   Примеры
  </h2>
  <h3>
   Пример 1
  </h3>
  <p>
   <strong class="sc-ce364f8a-0 bmdYeF">
    Ввод:
   </strong>
  </p>
  <pre><div class="code-block-wrapper"><pre class="code-common code-block prism"><code class="language-python"><span class="token number">5</span> <span class="token number">2</span> <span class="token number">9</span> <span class="token number">1</span> <span class="token number">7</span> </code></pre></div></pre>
  <p>
   <strong class="sc-ce364f8a-0 bmdYeF">
    Вывод:
   </strong>
  </p>
  <pre><div class="code-block-wrapper"><pre class="code-common code-block prism"><code class="language-python"><span class="token number">5</span> </code></pre></div></pre>
  <p>
   <strong class="sc-ce364f8a-0 bmdYeF">
    Объяснение:
   </strong>
   После сортировки список становится [1, 2, 5, 7, 9]. Средний элемент — 5.
  </p>
  <h3>
   Пример 2
  </h3>
  <p>
   <strong class="sc-ce364f8a-0 bmdYeF">
    Ввод:
   </strong>
  </p>
  <pre><div class="code-block-wrapper"><pre class="code-common code-block prism"><code class="language-python"><span class="token number">4</span> <span class="token number">1</span> <span class="token number">8</span> <span class="token number">5</span> </code></pre></div></pre>
  <p>
   <strong class="sc-ce364f8a-0 bmdYeF">
    Вывод:
   </strong>
  </p>
  <pre><div class="code-block-wrapper"><pre class="code-common code-block prism"><code class="language-python"><span class="token number">4.5</span> </code></pre></div></pre>
  <p>
   <strong class="sc-ce364f8a-0 bmdYeF">
    Объяснение:
   </strong>
   После сортировки список становится [1, 4, 5, 8]. Два средних элемента — 4 и 5, а их среднее значение — 4.5.
  </p>
