   FizzBuzz
  </h1>
 </div>
 <div class="question-text">
  <p>
   FizzBuzz — это классическая задача программирования, часто используемая на собеседованиях. Задача проста, но даёт возможность продемонстрировать ясное мышление и практики кодирования.
  </p>
  <p>
   Ваша задача — реализовать функцию
   <span class="code code-line code-line--blue-color">
    fizzbuzz
   </span>
   , которая возвращает массив строк в соответствии с правилами FizzBuzz.
  </p>
  <h2>
   Формат ввода
  </h2>
  <p>
   Функция получает целое число
   <span class="code code-line code-line--blue-color">
    n
   </span>
   .
  </p>
  <h2>
   Формат вывода
  </h2>
  <p>
   Функция должна возвращать массив строк, где:
  </p>
  <ul>
   <li>
    Для чисел, делящихся на 3, массив содержит "Fizz" вместо числа
   </li>
   <li>
    Для чисел, делящихся на 5, массив содержит "Buzz" вместо числа
   </li>
   <li>
    Для чисел, делящихся и на 3, и на 5, массив содержит "FizzBuzz" вместо числа
   </li>
   <li>
    Для остальных чисел массив содержит строковое представление самого числа
   </li>
  </ul>
  <h2>
   Ограничения
  </h2>
  <ul>
   <li>
    Входное значение
    <span class="code code-line code-line--blue-color">
     n
    </span>
    будет целым числом от 1 до 100.
   </li>
  </ul>
